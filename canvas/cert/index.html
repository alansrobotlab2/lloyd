Installation

Chrome / Edge (Windows)
  1. Double-click the downloaded .crt file
  2. Click Install Certificate
  3. Choose Current User, click Next
  4. Select Place all certificates in the following store
  5. Click Browse, select Trusted Root Certification Authorities
  6. Click Next, then Finish
  7. Restart the browser
Chrome / Edge (macOS)
  1. Double-click the downloaded .crt file (opens Keychain Access)
  2. It will appear in the login keychain
  3. Double-click the cert, expand Trust
  4. Set When using this certificate to Always Trust
  5. Close and enter your password
  6. Restart the browser
Chrome (Linux)
  1. Open chrome://settings/certificates
  2. Go to the Authorities tab
  3. Click Import, select the .crt file
  4. Check Trust this certificate for identifying websites
  5. Click OK, restart the browser
Firefox (all platforms)
  1. Open about:preferences#privacy
  2. Scroll to Certificates, click View Certificates
  3. Go to the Authorities tab
  4. Click Import, select the .crt file
  5. Check Trust this CA to identify websites
  6. Click OK, restart the browser
Safari / iOS
  1. Tap the download link on your device
  2. Go to Settings → General → VPN & Device Management
  3. Tap the downloaded profile and install it
  4. Go to Settings → General → About → Certificate Trust Settings
  5. Enable full trust for the certificate
Android
  1. Tap the download link
  2. Go to Settings → Security → Encryption & credentials
  3. Tap Install a certificate → CA certificate
  4. Select the downloaded file
